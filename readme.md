## 获取环境变量
process.env 是读取系统环境变量的，比如你在启动服务的时候，设置环境变量为production或者development，那么在程序里面就可以通过process.env.ENVNAME获取，因为你在node命令窗口启动时没有设置相关的环境变量，所以就没办法获取到了，你可以试一下NODE_ENV="development" node来启动命令窗口，然后应该就可以获取到了！
